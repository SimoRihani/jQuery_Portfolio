Site de recette Cuisin'Eirb - SGBD



Dans le cadre des études au semsestre 7, l'un des projets était la réalisation d'une base de données pour gérer des recettes.





















Présentation

Cuisin'Eirb est un site de recettes élaboré en projet informatique de Système de gestion de base de données (SGBD).
Ce site constitue une base de données pour gérer des recettes.
C'est un site destiné aux élèves de l'ENSEIRB-MATMECA, et qui offre les possiblités :
  • de proposer des recettes.
  • de consulter les recettes disponibles, de les commenter et de les noter.

Approche de développement

Le développement s'est fait en équipe de 3 dans une approche :
  • en parallèle : avancement sur la partie théorique de la base de données et développement web.
